🚀 Getting Started (Read Order Guide)

Follow this order to correctly set up and run the local AI environment.

---

📌 Step-by-Step Guide

1️⃣ Requirements

Start here to ensure your device and software are ready:

👉 "REQUIREMENTS.md"

---

2️⃣ Termux & Initial Setup

Install Termux and prepare the environment:

👉 "setup/README.md"

---

3️⃣ Build llama.cpp

Compile the AI engine:

👉 "setup/BUILD.md" (or section inside setup guide)

---

4️⃣ Download Models

Get the required AI models (GGUF format):

👉 "setup/MODEL_DOWNLOAD.md"

---

5️⃣ Create & Use Scripts

Set up scripts to control servers:

👉 "setup/SCRIPTS.md"

---

6️⃣ Launch Server

Run your AI model locally:

👉 Use:

./start_ai.sh

---

7️⃣ Connect to Acode

Integrate with Acode IDE:

👉 "setup/ACODE_SETUP.md"

---

🔄 Workflow Summary

Requirements → Setup → Build → Models → Scripts → Server → Acode

---

⚠️ Important Notes

- Follow steps in order to avoid errors
- Do not skip the build step
- Ensure models are downloaded before running scripts
- Only one model runs on port "8080" at a time

---

✅ Final Result

After completing all steps, you will have:

- Local AI server running
- Coding + chat models ready
- Acode IDE connected to your local AI

---

🎯 Goal

A fully offline AI development environment running on your device.
